# react-native-scale-size
```
React Native屏幕适配工具
```

### Getting started
#### 1.安装
```
$ npm i react-native-scale-size --save
```
#### 2.导入
```
import ScaleSize from 'react-native-scale-size';
```

### Example
```
const styles = StyleSheet.create({
  content: {
    width: ScaleSize.scaleSize(100),
    fontSize: ScaleSize.scaleText(18)
  }
})
```
